NeuroPlus Management Software - Offline Setup

1. Open the "ManagementSoftware" folder.
2. Double-click "index.html" to run it in your browser (Chrome recommended).
3. First launch: set your password.
4. Add patient details and save. Data is stored locally (no internet required).
5. To clear data, open Chrome > Developer Tools > Application > Local Storage > Clear.

All data stays private on your device.
